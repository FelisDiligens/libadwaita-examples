## load-template in Vala

Loads a template XML from a compiled gresource file.

Run inside `load-template/vala/`:

```bash
# Set up the project build folder (only required once):
meson setup build
# Build the project:
ninja -C build
# Run the program:
./build/src/load-template-vala
```
